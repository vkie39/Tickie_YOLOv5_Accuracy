yolo v5에서 사용하는 손실함수들

yolo v5는 객체 탐지 모델이어서 이 모델의 손실함수는 3가지가 제일 중요한 것 같다고 함
BBox (Bounding Box) Regression Loss
Objectness Loss
Class Loss

1. Bounding Box (BBox) Regression Loss
예측된 경계 상자와 실제 경계 상자(Ground Truth) 사이의 차이 측정. 주로 IoU (Intersection over Union) 사용.
YOLOv5에서는 **Complete Intersection over Union (CIoU)**와 같은 발전된 IoU 변형을 사용 가능. CIoU는 위치, 크기, 중심점을 고려하여 더 정교한 손실을 계산한다고 함. 우리가 하기에는 어렵겠지..?

수식: (chatgpt에 찾아보셈. 짜피 못 알아들음)

2. Objectness Loss:
예측된 상자가 객체를 포함하는지 여부에 대한 신뢰도(confidence) 측정합니다.
긍정 샘플 (Positive sample, 실제 객체가 있는 곳)과 부정 샘플 (Negative sample, 객체가 없는 곳)을 구별하는데 사용됨
일반적으로 이진 교차 엔트로피(Binary Cross-Entropy) 손실 사용
